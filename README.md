# Docker Dev

**System-wide development container**

Docker Dev is a persistent dev container that helps you to protect yourself from potentially malicious programs, like NPM dependencies. It runs continuously, providing faster access and better resource efficiency compared to per-project dev containers.

**Lean how to set it up at [zwyx.dev/blog/docker-dev](https://zwyx.dev/blog/docker-dev)**
